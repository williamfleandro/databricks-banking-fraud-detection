# Databricks Banking Fraud Detection MLOps Pipeline

Projeto de MLOps para detecção de fraudes bancárias usando Databricks, Unity Catalog, Delta Lake, MLflow, Feature Engineering, Data Quality, Drift Monitoring e Databricks Asset Bundles.

## Status

Projeto em construção.

## Objetivo

Construir uma esteira completa para classificação de transações fraudulentas, contemplando:

- Ingestão de dados
- Camadas Bronze, Silver e Gold
- Data Quality
- Feature Engineering
- Treinamento de modelos
- MLflow Tracking
- Model Registry
- Batch Inference
- Monitoramento de drift
- Retreinamento
- Deploy em DEV, ACC e PROD
